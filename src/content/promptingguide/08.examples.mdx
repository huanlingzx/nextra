# **实用提示词范例 —— 提升您的日常工作效率**

掌握了核心原则和基本技巧后，让我们来看一些在日常工作和学习中非常实用的提示词范例。这些范例会尝试融合多种原则和技巧，以期获得更佳的输出效果。

* **撰写邮件**
  * *任务场景*：您需要给一位潜在客户发送一封邮件，介绍公司新推出的一款项目管理软件。
  * *提示词示例*：

    ```markdown
    请扮演一位专业的客户成功经理（角色扮演）。
    我需要你为我们的新产品“高效协作云平台”（上下文：产品名称）撰写一封推广邮件。
    目标客户是中小型科技创业公司的项目经理（清晰具体：目标受众）。
    邮件的核心内容应突出这款平台如何帮助他们：
    1. 提升团队沟通效率；
    2. 实时追踪项目进展；
    3. 简化任务分配与管理。（清晰具体：核心价值点，以列表形式更清晰）
    邮件风格应专业、简洁且具有说服力（清晰具体：风格）。
    篇幅请控制在300-400字之间（约束：篇幅）。
    邮件开头应有个性化的称呼（例如，“尊敬的[客户姓名]经理”），结尾需要包含一个明确的行动号召，例如邀请他们注册免费试用或预约产品演示（清晰具体：行动号召与细节）。
    请提供邮件的标题和正文。
    ```

  * *设计思路剖析*：这个提示词综合运用了角色扮演（客户成功经理）、提供了充足的上下文（产品名称、目标客户画像）、明确了核心内容和价值点（通过列点方式）、设定了风格和篇幅约束，并指明了邮件结构的关键要素（称呼、行动号召）。
* **文本摘要与信息提取**
  * *任务场景*：您拿到一份长达 10 页的市场调研报告，需要快速了解其核心发现和关键数据。
  * *提示词示例*：

    ```markdown
    我将粘贴一份关于“2024年全球电动汽车市场趋势”的调研报告节选（上下文：文档主题）。
    请你仔细阅读以下文本，然后帮我完成以下任务：
    1. 用不超过200字总结这份报告的核心观点（清晰具体：任务1，约束：篇幅）。
    2. 提取报告中提到的未来三年市场增长率预测的具体数字（清晰具体：任务2，提取内容）。
    3. 列出报告中提及的影响市场增长的三个主要驱动因素（清晰具体：任务3，提取内容，约束：数量）。
    请将总结、增长率预测和驱动因素分别清晰地列出（约束：格式）。

    [此处粘贴报告文本]
    ```

  * *设计思路剖析*：该提示明确了输入文本的来源和主题，将复杂的摘要和提取任务分解为三个具体的子任务，并对每个任务的输出形式和内容都给出了清晰的指示和约束。
* **内容创作与头脑风暴**
  * *任务场景*：您正在为一个新的健康生活方式 App 策划一系列博客文章，需要一些选题灵感。
  * *提示词示例*：

    ```markdown
    我正在为一个专注于“都市年轻人健康生活方式”的App（上下文：产品定位和目标用户）进行内容规划。
    请你扮演一位资深健康内容策划师（角色扮演）。
    为我头脑风暴至少5个原创的博客文章选题（清晰具体：任务和数量）。
    这些选题应该围绕以下一个或多个主题：
    - 办公室简易健身技巧
    - 健康饮食与快速食谱
    - 缓解工作压力与焦虑的方法
    - 改善睡眠质量的建议（清晰具体：主题范围）
    选题应具有吸引力，能够引起22-30岁都市白领的兴趣（清晰具体：目标受众和风格要求）。
    请同时为每个选题附上一句简短的（不超过30字）内容梗概（约束：附加要求和篇幅）。
    ```

  * *追加提示 (迭代优化)*： “非常好！基于你刚才提供的选题，能否针对‘缓解工作压力与焦虑的方法’这个主题，再深入构思 3 个更侧重于‘正念练习’和‘利用碎片时间放松’的具体文章角度？” 
  * *设计思路剖析*：此提示通过角色扮演设定了专业视角，明确了产品背景、目标用户和内容主题范围。它要求具体数量的选题，并对选题的吸引力和附加梗概提出了要求。后续的追加提示则体现了迭代优化的过程，针对某个方向做更深入的挖掘。
* **辅助学习与知识问答**
  * *任务场景*：您想理解一个比较抽象的经济学概念，比如“机会成本”。
  * *提示词示例*：

    ```markdown
    请你扮演一位风趣幽默的经济学老师（角色扮演）。
    向一个完全没有经济学基础的大学生（清晰具体：目标受众）解释什么是“机会成本”（清晰具体：主题）。
    请务必使用至少一个生动的、贴近日常生活的例子来阐述这个概念（清晰具体：解释方式和要求）。
    解释完之后，请用“让我们一步一步地思考”作为引导（零样本CoT引导），分析一下，如果这位大学生选择花一个下午去看电影，他可能放弃了哪些机会成本？
    最后，请提出1-2个简单的问题，帮助我检验是否理解了这个概念（清晰具体：互动要求）。
    ```

  * *设计思路剖析*：这个提示不仅设定了角色和目标受众，还明确要求使用生活化例子，并引入了零样本思维链提示来引导模型进行分析。最后的提问要求则增加了互动性和检验学习效果的环节。
* **简单代码生成与解释**
  * *任务场景*：您需要一个 Python 函数来检查一个单词是否是回文（正读反读都一样）。
  * *提示词示例*：

    ```markdown
    请用Python 3（清晰具体：语言和版本）编写一个函数，函数名为 `is_palindrome`。
    这个函数接受一个字符串作为输入参数（清晰具体：输入）。
    函数的功能是判断输入的字符串是否为回文字符串（忽略大小写和非字母数字字符）（清晰具体：功能和处理细节）。
    如果是回文，函数应返回 True，否则返回 False（清晰具体：输出）。
    请在代码中为关键的逻辑步骤添加清晰的注释（约束：代码注释）。
    最后，请提供至少两个使用这个函数的测试示例，一个返回True，一个返回False（约束：测试示例）。
    ```

  * *设计思路剖析*：该提示明确了编程语言、函数名、输入输出、核心功能逻辑（包括了对大小写和特殊字符的处理），并要求添加注释和测试用例，这些都是高质量代码生成提示的关键要素。

通过观察以上这些“实用提示词范例”，我们可以发现一个共同点：**优秀的提示词往往不是单一原则或技巧的简单应用，而是多种元素的巧妙融合**。例如，在撰写邮件的提示中，我们既指定了角色（角色扮演），又提供了产品和客户的背景信息（上下文），明确了邮件的风格、篇幅和核心内容（清晰具体与约束）。在学习新概念的提示中，角色扮演、目标受众定义、解释方式要求、思维链引导以及互动提问等多种元素被结合起来，共同构成了一个丰富而指向明确的指令。

这给我们的启示是，不应将前面介绍的各项原则和技巧视为孤立的“招式”，而应将其看作一个工具箱，根据具体任务的需求，灵活地组合使用这些工具，才能打造出真正强大、能够充分发挥 LLM 潜能的提示词。这种组合应用的思维，是从提示词新手向高手进阶的关键一步。
