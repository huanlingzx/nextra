# 引言

大语言模型 (Large Language Models, LLMs) 作为一项革命性的人工智能技术，正在迅速改变我们工作、学习以及与信息交互的方式。

它们如同拥有强大理解和创造能力的智能助手，能够协助我们完成从文本生成到复杂问答等多种任务。

本教程主要用于入门，过阅读本教程，您将能够轻松理解 LLM 的基本概念，并重点掌握提示词工程 (Prompt Engineering) 这一核心技能，可以利用 LLM 提升日常工作与学习效率。

本教程将引导您从认识 LLM 是什么、它们能做什么，到理解它们的基本工作原理。随后，我们将深入探讨提示词工程的艺术与科学，学习如何通过精心设计的指令与 LLM 高效沟通。最后，我们还会讨论如何负责任地使用 LLM，了解并应对其潜在的局限性。希望这段学习旅程能为您打开一扇通往更高效、更智能工作方式的大门。
